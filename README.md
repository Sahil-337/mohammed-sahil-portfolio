# Mohammed Sahil Portfolio

A clean React + Vite portfolio website for Mohammed Sahil, designed for Data Science, Data Analyst, BI Analyst, ML Analyst, Product Analyst, Healthcare Data Analyst, Analytics Engineer, and Strategy/Operations Analytics job applications.

## Stack

- React
- Vite
- Tailwind CSS
- lucide-react icons

## Local Setup

1. Install Node.js from https://nodejs.org if you do not already have it.
2. Open a terminal in this project folder.
3. Run:

```bash
npm install
npm run dev
```

4. Open the local URL shown in your terminal, usually:

```text
http://localhost:5173
```

## Resume PDF

The resume file should be placed here:

```text
public/Mohammed_Sahil_Resume.pdf
```

If you replace the resume later, keep the same file name so the website download buttons keep working.

## Build for Deployment

```bash
npm run build
```

Vite will create a `dist/` folder.

## Publish on Vercel

1. Create a GitHub repository, for example `mohammed-sahil-portfolio`.
2. Upload/push this project to GitHub.
3. Go to https://vercel.com and sign in with GitHub.
4. Click **Add New Project**.
5. Import your GitHub repository.
6. Framework preset should auto-detect as **Vite**.
7. Build command: `npm run build`.
8. Output directory: `dist`.
9. Click **Deploy**.
10. After deployment, copy the Vercel URL and add it to your resume and LinkedIn.

## Publish on Netlify

1. Create a GitHub repository and upload/push this project.
2. Go to https://netlify.com and sign in.
3. Click **Add new site** > **Import an existing project**.
4. Connect GitHub and select the repository.
5. Build command: `npm run build`.
6. Publish directory: `dist`.
7. Click **Deploy site**.
